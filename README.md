### 依赖安装
```
pip install fastapi
pip install uvicorn[standard]
pip install python-multipart

pip install SQLalchemy
pip install pymysql
```

### 项目特点
```
1.大文件上传
2.统一接口返回值
3.全局异常捕获
4.全局跨域处理
5.请求拦截
6.数据库CRUD
7.静态资源访问
8.WebSokect即时通讯
9.标准RestfulApi前后端分离项目
10.Swagger接口管理
11.Jwt用户登录安全认证
12.AES接口加密校验
```


### 目录介绍
```
common          公共的类和方法
config          配置
db              数据库
handler         全局异常捕获
middleware      中间件
model           实体类
router          接口
im              即时通讯
service         服务
static          静态资源文件
util            工具类
main.py         工程入口
```
